================================================================================

üêö ENTER DJANGO CONTAINER

  python manage.py createsuperuser

================================================================================
                        DATABASE ACCESS
================================================================================

üíæ ENTER POSTGRESQL
--------------------
docker compose exec db psql -U postgres -d dzfellah

Inside PostgreSQL:
  \dt                     # List all tables
  \d users               # Describe users table
  SELECT * FROM users;   # Query users
  \q                     # Exit


üì§ BACKUP DATABASE
------------------

üì• RESTORE DATABASE
-------------------
